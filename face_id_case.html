Face ID

Project Idea

The idea was to make a way to automatically detect who was at a camera with the ability to detect other people if fed with a dataset of picture. This was so I could later implement this at future projects.

Core Technologies
  • Python
Features
  • Identifies faces drawing a blue box around it
  • Can detect eyes
  • Gives out a percentage of its certainty
  • live certainty percentage changes

I made a face id system because I wanted to learn more about OpenCv and get to understand how it can be used to train algorithms and recognise it. The most critical part was tweaking variables to ensure it wont give out false positives or false negatives.

Challenges

I noticed that some of the times the percentage would go above 100% if I prevented this it resulted in less accurate
estimates. This could be seen as some of the times when detecting Elon Musk its percentage is over 100. In the
future, this could potentially be solved by having a larger set of pictures which the algorithm can learn from.
